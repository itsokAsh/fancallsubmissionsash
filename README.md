## 📱 About Fancall

Fancall is a creator-centric platform that enables fans to connect with creators instantly through paid 1:1 audio and video calls — without the noise, delays, or chaos of DMs.

Unlike passive content platforms, Fancall is built for high-intent, real-time interactions, where:

- **Fans** get direct access to creators they genuinely care about
- **Creators** monetise their time and expertise, not ads or reach
- **Every interaction** is intentional, time-bound, and valuable

> **Fancall is live, with real users, real creators, and real interaction data.**  
> This hackathon is not theoretical. You'll be working on a real product problem we actively want to ship.

## 🗺️ Current User Journeys

### Fan Journey

1. Sign up via email or phone
2. Select at least 5 interest categories
3. Browse creators by category or search manually
4. View profiles, book audio/video calls
5. Add creators to favourites

> **Discovery is currently manual**

### Creator Journey

1. Sign up via Instagram or YouTube
2. Choose a category / niche
3. Set availability, call type, and pricing
4. Appear in category listings or search results

> **Visibility is currently non-personalised**

👉 **Your task is to add the recommendation layer into this existing experience.**

## 📊 Data You'll Get

To make this realistic, we'll provide structured datasets, including:

### Fan Data
- Age, location
- Interest categories
- Search logs
- Call booking history
- Frequency & recency of usage
- Favourite activity

### Creator Data
- Category / niche
- Location
- Call availability & pricing
- Engagement indicators
- Historical fan interactions

### Interaction Data
- Fan ↔ creator call history
- Search → click → booking behaviour
- Time-based activity patterns

### You're Encouraged To:
- ✅ Create derived signals
- ✅ Apply weighting or ranking logic
- ✅ Propose additional ideas using this data
